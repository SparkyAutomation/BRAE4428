The Artificial Neuron (Perceptron)

The simplest building block. It takes multiple inputs (e.g., soil moisture and sunlight), computes a weighted sum (dot product), and applies a harsh binary threshold: 1 (Water) or 0 (Don't Water).

Activation Function: Step
Neuron Structure
1. The Weighted Sum (z)
z = (w₁ · x₁) + (w₂ · x₂) + b
2. The Step Activation
ŷ = 1 if z ≥ 0  |  ŷ = 0 if z < 0
Crucial Insight: The Dot Product. Think of the weight vector [w₁, w₂] as a learned pattern. The calculation (w₁·x₁ + w₂·x₂) is a dot product, which mathematically checks the similarity between the input data and the neuron's learned pattern. If they align strongly, the dot product is highly positive, and the neuron fires!

The Sigmoid Neuron

A perceptron's rigid "step" shatters the loss landscape. To use modern Gradient Descent, we need smoothness. We swap the step function for the Sigmoid function, outputting a continuous probability instead of a hard binary.

Activation Function: Sigmoid
Neuron Structure
The Sigmoid Activation Function
a = σ(z) = 1 / (1 + e−z)
Calculus-Friendly: Because the sigmoid curve is smooth and continuous everywhere, we can take its derivative. This is the crucial mathematical trick that allows the neuron to calculate exactly which direction to adjust its weights.

The Neural Network (Multi-Layer Perceptron)

A single neuron can only draw a straight line. What if our data has a complex, cross-shaped boundary? By stacking neurons into hidden layers, they combine their straight lines into complex, non-linear shapes.

Network Architecture (2 → 4 → 1)

It takes two sensor inputs, passes them through a hidden layer of 4 sigmoid neurons, and combines them into 1 final output prediction.
